# Car-Price-Prediction-Project-CodeAlpha-Task-3
Machine Learning project that predicts used car selling prices using Random Forest Regression. Includes data preprocessing, feature engineering, exploratory data analysis, model evaluation, and price prediction.

## Overview

This project predicts the selling price of used cars using Machine Learning techniques. A Random Forest Regression model is trained on car-related features such as age, present price, kilometers driven, fuel type, transmission type, and ownership history to estimate car prices accurately.

## Objectives

* Analyze factors affecting car prices.
* Perform data preprocessing and feature engineering.
* Build a machine learning model for price prediction.
* Evaluate model performance using regression metrics.
* Predict the selling price of used cars.

##  Model Used

**Random Forest Regressor**

The model was trained to learn patterns from historical car data and predict selling prices based on vehicle characteristics.

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

##  Key Findings

* Present price significantly influences selling price.
* Older cars generally have lower resale values.
* Driven kilometers affect car prices.
* Fuel type and transmission contribute to price prediction.
* Random Forest Regression provides strong predictive performance.
